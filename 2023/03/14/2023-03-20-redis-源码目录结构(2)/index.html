redis-源码目录结构-(2)

发表于2023-03-14|更新于2023-03-20|redis
|阅读量:

目录结构

  • dept：包含了redis依赖的第三方代码库
    • 包括redis的c语言版本客户端代码hiredis
    • jemalloc内存分配器代码
    • readline功能的替代代码linenoise
    • lua代码
  • src：功能模块的代码文件，也是redis源码的重要组成部分
    • modules：示例代码
    • 各功能模块源码
  • test：测试模块
    • unit：单元测试
    • cluster：功能测试
    • sentinel：哨兵功能测试
    • integration：主从复制功能测试
  • utils目录
    • create-cluster：创建集群工具代码
    • hashtable：rehash过程可视化代码
    • hyperloglog：误差率计算和展示代码
    • lru：lru算法效果展示

&emsp;&emsp;除了上述目录结构以外，还有一些服务器实例相关的功能模块以及对应的代码

  • tcp Socket
    • anet.h
    • anet.c
  • 客户端实现
    • networking.c
  • 初始化、主控制流程
    • server.h
    • server.c
  • 网络通信框架
    • ae.h
    • ae.c
    • ae_epoll.c
    • ae_evport.c
    • ae_kqueue.c
    • ae_select.c

&emsp;&emsp;数据类型以及对应操作的一些功能源码

  • sds：string类型
    • sds.h/sds.c/sdsalloc.h
  • 双向链表：list
    • adlist.h/adlist.c
  • 压缩列表：list、hash、sorted set
    • ziplist.h/ziplist.c
  • quickList：list、hash、sorted set
    • quicklist.h/quicklist.c
  • 整数列表：set
    • intset.h/intset.c
  • zipmap：hash
    • zipmap.h/zipmap.c
  • 哈希表：hash
    • dict.h/dict.c
  • hyperloglog：hyperloglog
    • hyperloglog.c
  • geohash：geo
    • geo.h/heo.c
    • geohash.h/geohash.c
    • genhash_helper.h
    • genhash_helper.c
  • 位图：位图
    • bitops.c
  • stream：时序数据
    • stream.h/stream.c

&emsp;&emsp;以及数据库相关操作:

  • 数据库操作：db.c
  • 内存分配：zmalloc.h/zmalloc.c
  • 内存回收：expire.c/lazyfree.c
  • 数据替换：evict.c

数据类型：

  • String（t_string.c、sds.c、bitops.c）
  • List（t_list.c、ziplist.c）
  • Hash（t_hash.c、ziplist.c、dict.c）
  • Set（t_set.c、intset.c）
  • Sorted Set（t_zset.c、ziplist.c、dict.c）
  • HyperLogLog（hyperloglog.c）
  • Geo（geo.c、geohash.c、geohash_helper.c）
  • Stream（t_stream.c、rax.c、listpack.c）

全局：

  • Server（server.c、anet.c）
  • Object（object.c）
  • 键值对（db.c）
  • 事件驱动（ae.c、ae_epoll.c、ae_kqueue.c、ae_evport.c、ae_select.c、networking.c）
  • 内存回收（expire.c、lazyfree.c）
  • 数据替换（evict.c）
  • 后台线程（bio.c）
  • 事务（multi.c）
  • PubSub（pubsub.c）
  • 内存分配（zmalloc.c）
  • 双向链表（adlist.c）

高可用&集群：

  • 持久化：RDB（rdb.c、redis-check-rdb.c)、AOF（aof.c、redis-check-aof.c）
  • 主从复制（replication.c）
  • 哨兵（sentinel.c）
  • 集群（cluster.c）

辅助功能：

  • 延迟统计（latency.c）
  • 慢日志（slowlog.c）
  • 通知（notify.c）
  • 基准性能（redis-benchmark.c）
